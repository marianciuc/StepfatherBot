# ClosenessBot
**Discord** bot for creating private rooms

Some basic bot commands are:
+ ```help```
+ ```private add <category id> <channel id>```
+ ```connect```
+ ```private limit <new limit>```
+ ```flip```
+ ```rand <max value>```
